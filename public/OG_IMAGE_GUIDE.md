# OG 이미지 가이드

## 필요한 이미지
- **파일명**: `og-image.png`
- **크기**: 1200 x 630 px
- **위치**: `/public/og-image.png`

## 이미지 내용 제안
1. **배경**: 그라데이션 (보라색-핑크)
2. **로고**: PetCrown 로고 + 왕관 아이콘
3. **텍스트**:
   - 메인: "펫크라운" (큰 글씨)
   - 서브: "우리 반려동물이 세상에서 제일 귀여워요! 💕"
   - 설명: "투표하고 랭킹 1위에 도전하세요! 🏆"
4. **장식**: 귀여운 강아지/고양이 일러스트 또는 사진

## 제작 도구
- Canva (https://www.canva.com/)
- Figma (https://www.figma.com/)
- Photoshop
- 온라인 OG 이미지 생성기

## 참고사항
- 텍스트는 중앙에 배치하고 가독성 확보
- 밝고 친근한 색상 사용
- 모바일에서도 잘 보이도록 큰 글씨 사용
- 브랜드 컬러(보라색, 핑크) 활용
